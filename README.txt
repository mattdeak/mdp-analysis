Getting the Code:
-------------------------
Clone the git repository at https://github.com/mattdeak/mdp-analysis:

`git clone https://github.com/mattdeak/mdp-analysis.git`

Setting up the Environment
------------------------
From a python 3 environment, run the command:
`pip install -r requirements.txt`

This will install all necessary libraries.

Running the Code
----------------------

To collect all the data, run the command:

`python run_experiments.py`

This will take a long time to run.

The plots can be generated by running:

`python generate_all_plots.py`

This will take less time, but still some time.
